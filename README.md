# Ariyan Hossain – Academic Portfolio

This repository contains my personal academic portfolio website.

## Tech Stack

- Next.js 14 (React, App Router)
- TypeScript
- Tailwind CSS
- React Icons
- EmailJS (for the contact form)

## Sections

- **Home / Resume** – About, work experience, education, selected research, and skills  
- **Projects** – Selected research and software projects with tech stack and links  
- **Achievements** – Academic awards, honors, and recognitions  
- **Teaching** – Courses I teach/taught, seminars/workshops, and academic services  
- **Contact** – Contact form for collaborations and inquiries

## Notes

- All content is defined in `data/profile.ts`  
- Static assets (images) live in the `public/` folder  
- The site is deployed to GitHub Pages via a GitHub Actions workflow
